## Installed software

+ **bspwm** - Window manager
+ **sxhkd** - Hotkey daemon
+ **compton** - Composite manager
+ **polybar** - Status bar
+ **termite** - Terminal emulator
+ **zsh + prezto** - Shell
+ **rofi** - Dynamic menu
+ **nautilus** - File manager
+ **atom** - Text editor

## GUI
+ **arc-grey-dark** - GTK theme
+ **paper** - Icon theme
+ **adwaita** - Cursor theme
